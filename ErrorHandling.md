# Error Handling
## A) Throws:
- This keyword used for the function which can be throw the error.

*Example:*
`func canThrowAnerror() throws {}`

## B) do, catch, try:
- To handle this type of error from the function try keyword is used under the do, catch block.

*Example:*
```
do{
  try canThrowAnerror()
} catch {

}
```
